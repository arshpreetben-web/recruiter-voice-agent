# Recruiter Voice Agent 🎙🤖

AI-based recruiter voice assistant that automates candidate screening using NLP and speech processing.

## 🚀 Features
- Resume analysis  
- Automated interview question generation  
- Voice-based interaction  
- Feedback generation using NLP models  
- Web interface using Flask  

## 🛠 Tech Stack
- Python  
- NLP  
- Flask  
- JavaScript  
- HTML  
- CSS  

## 📌 About
Collaboratively developed an intelligent voice-based recruitment assistant designed to simulate real interview workflows.
